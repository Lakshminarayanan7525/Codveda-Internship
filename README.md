# Codveda Internship - Java Development Tasks

This repository contains my Java solutions for the Codveda development internship, organized by task difficulty levels.

---

## 📂 Project Structure

* **Level 1 Tasks**
    * `BasicCalculator`: A terminal-based calculator featuring input loop functionality and division safety.
    * `NumberGuessingGame`: An interactive game tracking user attempts with randomized targets.
* **Level 2 Tasks**
    * `EmployeeManagementSystem`: A system demonstrating OOP principles to track employee details and salaries.
    * `SimpleBankingApplication`: A console application managing secure deposits, withdrawals, and balance inquiries.
* **Level 3 Tasks (Advanced)**
    * `BinarySearchTreeApp`: An implementation of a Binary Search Tree (BST) supporting dynamic node insertion, search, leaf/node deletions, and depth traversals (In-order, Pre-order, Post-order).
    * `LibraryManagementApp`: A complete CRUD application using **JDBC** and an embedded **SQLite database** to add, catalog, borrow, and return library books.

---

## 🛠️ Technologies Used
* **Language:** Java (JDK 23)
* **Database:** SQLite via JDBC
* **IDE:** IntelliJ IDEA
* **Version Control:** Git & GitHub
